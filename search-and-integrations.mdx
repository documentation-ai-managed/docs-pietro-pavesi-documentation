## Overview

Tacivo empowers you to search your captured institutional memory efficiently. Use advanced search features to query AI-ready documentation, uncover insights from expert conversations, and integrate with third-party tools. Export structured data to extend your knowledge moat across systems.

<Callout kind="tip">
  Start with simple keyword searches, then layer on filters for precise results.
</Callout>

## Advanced Search Features

Tacivo supports multiple search modalities to handle your growing knowledge base.

<Tabs>
  <Tab title="Keyword Search" icon="search">
    Perform full-text searches across conversations and extracted documents.
    
    Use operators like `AND`, `OR`, and quotes for phrases:
    
    ````javascript
    GET https://api.tacivo.com/v1/search?q="expert advice" AND topic:AI
    ````
  </Tab>
  <Tab title="Semantic Search" icon="zap">
    Leverage vector embeddings for context-aware results.
    
    ````javascript
    POST https://api.tacivo.com/v1/search/semantic
    {
      "query": "How to scale AI infrastructure?",
      "limit": 10,
      "filters": { "expert": "Alice Johnson" }
    }
    ````
  </Tab>
  <Tab title="Faceted Search" icon="filter">
    Refine results by metadata like date, expert, or topic.
    
    Add facets to your query for dynamic filtering.
  </Tab>
</Tabs>

## Querying AI-Ready Documentation

Follow these steps to query your structured knowledge.

<Steps>
  <Step title="Authenticate" icon="shield">
    Include your API key in the `Authorization` header.
  </Step>
  <Step title="Build Query" icon="code">
    Construct queries using Tacivo's query language.
    
    <CodeGroup tabs="REST,GraphQL">
    ````javascript
    fetch('https://api.tacivo.com/v1/knowledge/query', {
      method: 'POST',
      headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
      body: JSON.stringify({ q: 'scaling strategies' })
    });
    ````
    ````graphql
    query SearchKnowledge($q: String!) {
      results: search(query: $q, limit: 20) {
        id
        content
        relevance
      }
    }
    ````
    </CodeGroup>
  </Step>
  <Step title="Parse Results" icon="list">
    Handle paginated responses with metadata.
  </Step>
</Steps>

## Exporting Structured Data

Export your knowledge in standard formats for external use.

```mermaid
graph LR
    A[Query Knowledge] --> B[Apply Filters]
    B --> C[Select Format]
    C --> D[JSON/CSV Export]
    D --> E[Integrate Externally]
```

<Expandable title="Advanced Export Options" default-open="false">
  Customize exports with projections and transformations.
  
  <Request tabs="cURL,Node.js">
  ````bash
  curl -X POST https://api.tacivo.com/v1/export \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "query": "all conversations",
      "format": "json",
      "fields": ["id", "expert", "content"]
    }'
  ````
  ````javascript
  const fs = require('fs');
  const response = await fetch('https://api.tacivo.com/v1/export', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      query: 'recent experts',
      format: 'csv'
    })
  });
  const data = await response.text();
  fs.writeFileSync('knowledge.csv', data);
  ````
  </Request>
</Expandable>

<Callout kind="info">
  Exports respect your access controls, ensuring secure data sharing.
</Callout>
