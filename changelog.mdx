## Upgrading Tacivo

Follow these steps to upgrade your installation safely.

<Steps>
  <Step title="Backup Data" icon="database">
    Export your conversations and knowledge base.

    ```bash
    tacivo export --all --output=backup-$(date +%Y%m%d).zip
    ```
  </Step>
  <Step title="Update Package" icon="package">
    <Tabs>
      <Tab title="npm" icon="nodejs">
        ```bash
        npm update tacivo-sdk@latest
        ```
      </Tab>
      <Tab title="Docker" icon="docker">
        ```bash
        docker pull tacivo:latest
        docker-compose up -d
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Run Migrations" icon="database">
    ```bash
    tacivo migrate --env=production
    ```
  </Step>
  <Step title="Verify" icon="check-circle">
    Test key endpoints and search functionality.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Verify SDK version post-upgrade
  import { Tacivo } from 'tacivo-sdk';
  console.log(Tacivo.version); // Should log >=1.2.0
  ```
  ```python
  # Verify SDK version post-upgrade
  import tacivo
  print(tacivo.__version__)  # Should print >=1.2.0
  ```
</CodeGroup>
